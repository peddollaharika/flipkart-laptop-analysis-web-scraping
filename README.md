# Flipkart Laptop Market Analysis Using Web Scraping and Exploratory Data Analysis

## Project Overview

This project uses Web Scraping techniques to collect laptop data from Flipkart and performs Exploratory Data Analysis (EDA) to identify market trends, pricing patterns, and customer preferences. The project demonstrates how raw web data can be transformed into meaningful business insights.

## Business Problem Statement

Customers often find it difficult to compare laptops based on price, specifications, ratings, and brands available on e-commerce platforms. This project automates data collection and analysis to simplify laptop comparison and support data-driven decision-making.

## Objectives

* Scrape laptop data from Flipkart
* Store data in a structured format
* Clean and preprocess the collected data
* Extract important features using Regex
* Perform Exploratory Data Analysis (EDA)
* Generate business insights and recommendations

## Technologies Used

* Python
* Requests
* BeautifulSoup
* Pandas
* NumPy
* Regular Expressions (Regex)
* Matplotlib
* Seaborn

## Workflow

1. Data Collection using Web Scraping
2. Data Cleaning and Preprocessing
3. Feature Engineering using Regex
4. Exploratory Data Analysis (EDA)
5. Business Insights Generation

## Features Extracted

* Brand
* Processor
* RAM
* Storage
* Price
* Original Price
* Discount
* Ratings Count
* Reviews Count

## Key Insights

* Intel processors dominate the laptop market.
* Apple laptops have the highest average prices.
* HP, ASUS, Lenovo, and Acer have the highest number of listings.
* Most laptops belong to the budget and mid-range categories.
* Higher RAM and Storage configurations are associated with higher prices.
* Premium gaming and professional laptops create significant price outliers.

## Conclusion

The project successfully collected and analyzed laptop data from Flipkart using web scraping and EDA techniques. The analysis helped identify market trends, pricing patterns, processor preferences, and customer behavior, enabling better purchasing and business decisions.
